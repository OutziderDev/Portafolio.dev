# Mi Primer Post

¡Bienvenido a mi blog! Este es un ejemplo de cómo se ve el contenido **markdown** renderizado en mi portafolio, usando los colores de la paleta global.

## Introducción

El markdown es una forma ligera y fácil de escribir texto que se convierte automáticamente a HTML. Es perfecto para blogs, documentación y mucho más.

### Características principales

- Escribir sin necesidad de conocimientos de HTML
- Soporte para código, listas, enlaces e imágenes
- Renderizado automático en páginas web

## Ejemplos de formato

### Texto en negrita y cursiva

Puedes escribir texto en **negrita** o *cursiva*, o incluso ***ambos***.

### Listas
